How to use Simon's interface?

1. Launch from python environment

0. Verify that flask module is installed on your computer.
1. Run website.py program
2. In the Terminal, Ctrl + Leftclick on this link: http://127.0.0.1:5000/.
3. A window just opened in your web browser.

2. How to print

0. All units are in centimeters.
1. Mesure the height and width of your board.
2. Mesure the lengths of the ropes between the top corners and the prototype.
3. Choose where you want to print your drawing, the (0, 0) position is the center of the board.
4. Choose the height of your drawing.
5. Choose the drawing or the text you want to print.
6. Run the printing by clicking on the PRINT button.
7. Simon should be writting on the board, a turtle simulation opened itself on your computer.

Warning! You can not print a drawing and a text at the same time! Be sure to fill only one of the two option.

3. How to reuse Simon?

0. This section can be used if you have drawn something before and you want to draw something else.
1. Push the reset button.
2. Go to your IDE and stop the website.py program
3. Restart the website.py program
4. Go back to the interface and fill all the inputs.
5. Click the PRINT button again.

4. Print an image

1. Check if the image is in the available list, if not follow the instructions below.
2. Convert your image on this site into an SVG format.
3. Add your image.svg file in the SvgFiles folder.
4. Relaunch the interface and print it by writing your image name in the right input.
